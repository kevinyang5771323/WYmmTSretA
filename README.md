# 基于Python的语音和背景音乐分离算法及系统 python1183

## 项目概述

本项目是一款基于B/S架构开发的系统应用，旨在提供在线音频分离技术操作。用户可以通过系统实现音频上传、分离背景音乐与人声，并将分离后的音频保存。以下为系统的详细说明。

## 技术栈

### 后端

- 开发语言：Python
- 框架：Django
- Python版本：3.7.7
- 数据库：MySQL 5.7+
- 数据库工具：Navicat 11+
- 开发软件：PyCharm

### 前端

- Node.js
- Vue.js
- HTML

## 功能模块

### 系统登录界面

- 背景为动态图片
- 界面简洁，用户只需输入用户名和密码即可登录

### 音频分离功能

- 支持本地音频上传
- 上传后可执行分离操作，分离后得到原始音频、背景音乐音频和人声音频

### 个人信息查看界面

- 管理员权限下通过菜单栏图标查看
- 显示信息包括个人ID、用户名及角色

### 用户管理界面

- 显示系统用户列表

## 系统角色

- 普通用户：可进行音频上传和分离操作
- 管理员：除普通用户权限外，还可查看个人信息和进行用户管理

## 运行环境

- 后端：Python 3.7.7，Django，MySQL 5.7+
- 前端：兼容现代浏览器
